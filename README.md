# disclosure.ts (v2)
A [disclosure](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/) pattern implementation in TS. Using the `<details>` and `<summary>` element.
## Installation
```ts
import Disclosure from './disclosure';

const myDisclosure = new Disclosure(...);
```
