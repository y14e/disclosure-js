# disclosure.js (v2)
A [disclosure](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/) pattern implementation in JS. Using the `<details>` and `<summary>` element.
## Installation
```js
import Disclosure from './disclosure.js';

const myDisclosure = new Disclosure(...);
```
